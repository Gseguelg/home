01

Módulo powerfactory

Punto de entrada principal. Importar con import powerfactory.

Variable de módulo

Variable Tipo Descripción
powerfactory.__version__ str Versión instalada de PowerFactory, ej. "17.0.9"

Funciones de inicio

GetApplication()
powerfactory.GetApplication(
  [str username],
  [str password],
  [str commandLineArguments]
)
Crea y devuelve el objeto Application. Arranca PowerFactory si se llama desde externo. Mantener el objeto mientras se use PF.
→ Application | None
GetApplicationExt()
powerfactory.GetApplicationExt(
  [str username],
  [str password],
  [str commandLineArguments]
)
Igual que GetApplication() pero lanza powerfactory.ExitError (con .code) en caso de fallo. Recomendado para scripts de producción.
→ Application
Python
import sys
sys.path.append(r"C:\Program Files\DIgSILENT\PowerFactory 2021\Python\3.8")
import powerfactory

try:
    app = powerfactory.GetApplicationExt()
    # ... cálculos ...
except powerfactory.ExitError as error:
    print(f"Error código: {error.code}")

Application — Entorno y GUI

Control de la interfaz gráfica, progreso y comportamiento durante ejecución.

ExecuteCmd(command) None Ejecuta un comando como si se escribiera en el Input Window. Solo para pruebas.
PostCommand(command) None Encola un comando para ejecutar tras finalizar el script actual.
Rebuild([iMode]) None Redibuja el SLD o gráfico visible. iMode: 0=solo dibujar, 1=leer+dibujar, 2=leer+recalcular+dibujar.

Application — Matemáticas y Números Aleatorios

Generadores de números aleatorios, inversión de matrices y utilidades numéricas.

Método Retorna Descripción
RndSetup(seedAutomatic, [seed, rngType, rngNum]) None Inicializa generador. rngType: 0=Mersenne Twister (recomendado), 1=Linear Congruential, 2=Additive Lagged Fibonacci. rngNum=0: generador global.
RndUnifReal(min, max, [rngNum]) float Número aleatorio uniforme continuo en [min, max].
RndUnifInt(min, max, [rngNum]) int Número aleatorio uniforme entero en {min,...,max}.
RndNormal(mean, stddev, [rngNum]) float Número aleatorio distribución normal.
RndExp(rate, [rngNum]) float Número aleatorio distribución exponencial con tasa rate.
RndWeibull(shape, scale, [rngNum]) float Número aleatorio distribución Weibull.
RndGetSeed([rngNum]) int Retorna la semilla actual del generador rngNum.
RndGetMethod([rngNum]) str Nombre del método del generador rngNum.
InvertMatrix(realPart, [imaginaryPart]) int Invierte matriz real o compleja (Gauss-Jordan). 0=éxito, 1=singular. Resultado en la matriz de entrada.
GetMem([calculateDelta, inMegaByte]) int Consumo de memoria del proceso. Delta desde última llamada si calculateDelta=1.

Métodos comunes a todos los objetos de la BD para lectura y escritura de atributos.

Método Retorna Descripción
GetAttribute(name) any Valor del atributo en unidades mostradas. Lanza excepción si nombre inválido.
SetAttribute(name, value) None Establece valor de atributo.
HasAttribute(name) int 1 si el atributo existe en el objeto.
GetAttributeType(name) AttributeType INTEGER, DOUBLE, STRING, OBJECT, *_VEC, *_MAT, INTEGER64, INVALID.
GetAttributeUnit(name) str Unidad del atributo, ej. "km", "MW". None si nombre inválido.

ℹ️ Para acceder a atributos en unidades internas (no mostradas), usar app.SetAttributeModeInternal(1) antes de las llamadas. Usar app.DefineTransferAttributes(classname, "attr1,attr2") para configurar acceso en bloque con GetAttributes/SetAttributes.

ElmRes — Archivo de Resultados

Gestión completa de archivos de resultados de simulación: escritura, lectura e iteración.

Ciclo de Escritura

Método Descripción
AddVariable(element, varname) Agrega variable de element a monitorear. Alias: AddVars.
InitialiseWriting() Prepara el archivo para escritura manual.
Write(time, value) Escribe un valor en el tiempo dado.
WriteDraw() Escribe un "draw" (marca de actualización gráfica).
FinishWriting() Finaliza escritura. Alias: Close.
Flush() Vuelca buffer de memoria a disco. Necesario antes de leer si ya se escribió.
Clear() Limpia datos del archivo (mantiene definiciones de variables).

Ciclo de Lectura

Método Retorna Descripción
Load() int Carga datos del archivo en memoria para lectura.
Release() None Libera datos cargados en memoria.
GetNumberOfRows() int Número total de filas (instantes de tiempo).
GetNumberOfColumns() int Número total de columnas (variables monitoreadas).
FindColumn(obj, [varName / startCol]) int Índice de columna para objeto/variable. <0=no encontrado.
GetValue(obj, col, row) [int, float] Valor en fila/columna. Retorna [error, value].
GetObjectValue(row, col) float Valor en posición fila/columna directa.
GetObject(col) DataObject Objeto asociado a la columna col.
GetVariable(col) str Nombre de variable de la columna col.
GetUnit(col) str Unidad de la columna col.
GetDescription([col, short]) str Descripción de la columna.
FindMaxInColumn(column) [int, float] Máximo de la columna: [row, maxValue].
FindMinInColumn(column) [int, float] Mínimo de la columna: [row, minValue].
FindMaxOfVariableInRow(variable, row) [int, float] Máximo de variable en fila: [col, maxValue].
FindMinOfVariableInRow(variable, row) [int, float] Mínimo de variable en fila: [col, minValue].
SortAccordingToColumn(col) int Ordena filas según columna col.
SetAsDefault() None Establece como archivo de resultados por defecto.

Iteración (GetFirstValid / GetNextValid)

Método Descripción
GetFirstValidObject(row, [classNames, variableName, limit, limitOperator, limit2, limitOperator2]) Primera columna válida de un objeto en la fila. Soporta filtros por clase, variable y rango.
GetNextValidObject([classNames, variableName, limit...]) Siguiente columna válida (iterador interno).
GetFirstValidVariable(row, [variableNames]) Primera variable válida en la fila.
GetNextValidVariable([variableNames]) Siguiente variable válida.
GetFirstValidObjectVariable([variableNames]) Primera variable válida del objeto apuntado por el iterador.
GetNextValidObjectVariable([variableNames]) Siguiente variable válida del objeto actual.
ℹ️ Los operadores de límite: 0=todos válidos, 1=<, 2=≤, 3=>, 4=≥. Para limitOperator2: positivo=AND, negativo=OR.

Ejemplos de Uso Frecuente

Script externo — inicio básico

Python
import sys
sys.path.append(r"C:\Program Files\DIgSILENT\PowerFactory 2021\Python\3.8")
import powerfactory

app = powerfactory.GetApplication()
app.EchoOff()

app.ActivateProject("Mi_Proyecto")

# Obtener todos los terminales relevantes
terms = app.GetCalcRelevantObjects("*.ElmTerm")
for t in terms:
    uknom = t.GetAttribute("uknom")
    app.PrintInfo(f"{t.GetFullName()} → {uknom} kV")

app.EchoOn()

Python
ldf = app.GetFromStudyCase("ComLdf")
ldf.iopt_net = 0   # 0=AC, 1=DC
ierr = ldf.Execute()
if ierr == 0:
    app.PrintInfo("✓ Flujo de potencia exitoso")
else:
    app.PrintError("✗ Error en flujo de potencia")

Leer resultados de simulación dinámica

Python
res = app.GetFromStudyCase("ElmRes")
res.Load()

gen = app.GetCalcRelevantObjects("Gen1.ElmSym")[0]
col_p = res.FindColumn(gen, "s:P1")  # columna de potencia activa
col_t = res.FindColumn(res, "b:tnow")  # columna de tiempo

nrows = res.GetNumberOfRows()
for row in range(nrows):
    t = res.GetObjectValue(row, col_t)
    p = res.GetObjectValue(row, col_p)
    print(f"t={t:.3f}s  P={p:.4f}MW")

res.Release()

Modificar atributos con caché de escritura

Python
# Aumentar carga 10% con caché habilitada (más rápido)
app.SetWriteCacheEnabled(1)

for load in app.GetCalcRelevantObjects("*.ElmLod"):
    p_actual = load.GetAttribute("plini")
    load.SetAttribute("plini", p_actual * 1.10)

app.WriteChangesToDb()
app.SetWriteCacheEnabled(0)

Simulación dinámica completa (RMS)

Python
# 1. Flujo de potencia inicial
ldf = app.GetFromStudyCase("ComLdf")
ldf.Execute()

# 2. Condiciones iniciales
inc = app.GetFromStudyCase("ComInc")
inc.Execute()

# 3. Simulación
sim = app.GetFromStudyCase("ComSim")
sim.tstop  = 10.0   # [s]
sim.dtgrd  = 0.01   # paso de integración [s]
sim.iopt_sim = 0   # 0=RMS, 1=EMT
sim.Execute()

app.PrintInfo("Simulación completada.")
